# CODE_OF_CONDUCT.md

## Our Pledge

This is a **personal portfolio project**. While contributions are not expected, the purpose of this document is to ensure any interactions related to this repository remain professional, respectful, and constructive.

---

## Expected Behavior

- Be respectful in comments, issues, or discussions.
- Focus on constructive technical feedback.
- Avoid spam, harassment, or any offensive behavior.

---

## Unacceptable Behavior

- Any form of discrimination, harassment, or abuse.
- Posting sensitive information publicly.
- Off-topic discussions unrelated to development, learning, or code review.

---

## Reporting

If you notice inappropriate behavior or a security-related concern, please contact the maintainer **privately** at:

- Email: `dev@sandokan.cat`  

Do **not** post sensitive issues publicly.

---

## Enforcement

The maintainer reserves the right to **ignore, delete, or close** any issue or comment that violates these guidelines.  

> This project is primarily for learning and portfolio presentation. Professional conduct ensures the project is clear, accessible, and respected.
